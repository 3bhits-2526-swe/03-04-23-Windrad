Ultimate Windrad Simulator – VE Projekt 2026
Projektbeschreibung:
Der Ultimate Windrad Simulator ist ein Virtual‑Engineering‑Projekt, in dem der User eine 3D‑Landschaft mit drei Windrädern erkunden kann.
Jedes Windrad besitzt eine individuell einstellbare Windstärke, die sich auf die Drehgeschwindigkeit und Farbe auswirkt.
Beim Betreten des Interaktionsbereichs erscheint ein UI‑Fenster, über das der User die Windstärke anpassen kann.

MVP (Minimum Viable Product):
Startscene
- Buttons: Start, Profil, Einstellungen
- Start lädt die Hauptszene
3 Windräder
- Jedes Windrad hat:
 - eigene Windstärke
 - eigene Farbe abhängig von Windstärke
 - eigenen Trigger‑Bereich
Interaktions‑UI
- UI erscheint beim Betreten des Triggerbereichs
- Slider zur Einstellung der Windstärke
- Dynamische Änderung von Farbe & Drehgeschwindigkeit
Spielerbewegung
- Spieler kann frei durch die Landschaft laufen
- Kann jedes Windrad besuchen

Pitch
Unser VE‑Projekt simuliert einen interaktiven Windpark, in dem der User die Windstärke jedes Windrads individuell steuern kann. Durch ein intuitives UI, dynamische Farbänderungen und eine frei begehbare Landschaft entsteht ein realistisches und spielerisches Erlebnis, das technische Zusammenhänge sichtbar macht.
